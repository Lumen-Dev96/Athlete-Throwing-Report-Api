# sport-backend


## Enviroment

```
node >= 20.1.0
```

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
node server.js
```

### Compile and Minify for Production

```sh
npm run build
```

### Import Data

```
/storgae/information/{id}.txt
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
